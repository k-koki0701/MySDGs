# README

## 開発言語

- Ruby 2.6.5
- Ruby on Rails 5.2.5


## 就業Termの技術

- devise
- お気に入り機能
- フォロー機能


## カリキュラム外の技術

- ransack


## 実行手順

```
$ git clone git@github.com:k-koki0701/mysdgs.git
$ cd mysdgs
$ bundle install
$ rails db:create db:migrate
$ rails s
```
